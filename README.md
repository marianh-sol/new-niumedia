# Niumedia - Agencia de Marketing Digital

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/marianh-sol/new-niumedia)

Este es el sitio web oficial de **Niumedia**, optimizado para SEO técnico y diseñado con una estética moderna basada en *glassmorphism*.

## Características
- **Optimización SEO:** Metadatos únicos, etiquetas Open Graph y datos estructurados (JSON-LD) para mejorar el posicionamiento.
- **Tecnologías:** HTML5, CSS3, Bootstrap, jQuery.
- **Integraciones:** AOS (animaciones), Swiper/Owl Carousel, y configuración lista para HubSpot/Google Analytics.

## Despliegue en Render
Este repositorio incluye un archivo `render.yaml` que configura automáticamente el despliegue como un **Static Site**.

Para desplegar:
1. Haz clic en el botón **Deploy to Render** arriba.
2. Sigue las instrucciones en el dashboard de Render.

## Estructura
- `/dist`: Contiene los archivos finales del sitio listos para producción.
- `render.yaml`: Configuración de Blueprint para el despliegue automático.
